# Расчёт и диагностика трансформаторов

## Описание проекта

Это интерактивное учебное пособие, разработанное для студентов-электротехников и специалистов в области электроэнергетики. Сайт предоставляет комплексное введение в теорию трансформаторов, практические калькуляторы для расчётов и методы диагностики и контроля состояния силовых трансформаторов.

### Цели пособия

- **Обучение**: Предоставить студентам доступное и наглядное введение в теорию трансформаторов
- **Практика**: Помочь при выполнении расчётов параметров трансформаторов
- **Диагностика**: Познакомить с современными методами контроля состояния
- **Интерактивность**: Использовать интерактивные инструменты для лучшего понимания

## Теоретическое содержание

### Основы трансформаторов

**Определение**: Трансформатор — это статическое электромагнитное устройство, преобразующее переменный ток одного напряжения в переменный ток другого напряжения с помощью электромагнитной индукции.

**Классификация**:
- По напряжению (НН, СН, ВН, СВН)
- По мощности (малой, средней, большой)
- По числу фаз (однофазные, трёхфазные)
- По типу (масляные, сухие, измерительные, автотрансформаторы)

### Принцип работы

**Закон электромагнитной индукции Фарадея**:
```
E = 4.44 · f · N · Φₘ
```

**Коэффициент трансформации**:
```
K = U₁/U₂ = N₁/N₂ = I₂/I₁
```

### Потери энергии

**Постоянные потери (в стали)**:
- Потери на гистерезис
- Потери на вихревые токи
- Не зависят от нагрузки

**Переменные потери (в меди)**:
```
Pₘ = I² · R
```
- Пропорциональны квадрату нагрузки
- Растут с температурой

**КПД трансформатора**:
```
η = P₂ / (P₂ + Pст + Pм) · 100%
```

### Методы диагностики

1. **Тепловизионный контроль** — быстро выявляет горячие точки
2. **DGA** — анализ газов в масле для выявления скрытых дефектов
3. **Сопротивление изоляции** — оценка состояния диэлектрика
4. **Коэффициент абсорбции** — показатель поляризации изоляции
5. **Сопротивление обмоток** — выявление обрывов и коротких замыканий
6. **SFRA** — анализ механического состояния обмоток

## Установка и запуск

### Локальный запуск

1. Откройте файл `index.html` в веб-браузере
   - Можно просто дважды кликнуть на файл
   - Или перетащить файл в окно браузера

2. Или используйте локальный сервер:
   ```bash
   # Если установлен Python 3
   python -m http.server 8000
   
   # Если установлен Python 2
   python -m SimpleHTTPServer 8000
   ```
   Затем откройте `http://localhost:8000/index.html`

### Развёртывание на сервер

1. Загрузите все файлы на хостинг через FTP или Git
2. Убедитесь, что файлы находятся в одной директории
3. Откройте `index.html` через браузер

## Содержание файлов

### Калькуляторы и функции

#### Калькулятор 1: Коэффициент трансформации
```javascript
K = U₁ / U₂
N₂ = N₁ / K (если N₁ задано)
```

#### Калькулятор 2: Мощность и потери
```javascript
P₂ = U₂ · I₂  // Мощность нагрузки
Pₘ = I₂² · R  // Потери в меди
```

#### Калькулятор 3: КПД
```javascript
η = P₂ / (P₂ + Pст + Pм) · 100%
// Индикация: зелёный (>95%), жёлтый (>85%), красный (≤85%)
```

#### Калькулятор 4: Коэффициент абсорбции
```javascript
Ka = R₆₀ / R₁₅
// Оценка: отличное (≥1.5), допустимое (≥1.3), внимание (<1.3)
```
