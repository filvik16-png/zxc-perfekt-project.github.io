Интерактивные калькуляторы

Используйте калькуляторы ниже для практических расчётов параметров трансформаторов.

Калькулятор 1: Коэффициент трансформации

Результаты:

Коэффициент трансформации: K = 0

Число витков вторичной обмотки: N₂ = 0

Формула: K = U₁ / U₂ = N₁ / N₂

Калькулятор 2: Мощность и потери в меди

Результаты:

Мощность нагрузки: P₂ = 0 Вт

Потери в меди: P_м = 0 Вт

Формулы:

P₂ = U₂ · I₂

P_м = I₂² · R

Калькулятор 3: КПД трансформатора

Результаты:

КПД трансформатора: η = 0 %

Формула: η = P₂ / (P₂ + P_ст + P_м) · 100%

η > 95% — высокий КПД | η > 85% — средний КПД | η ≤ 85% — низкий КПД
